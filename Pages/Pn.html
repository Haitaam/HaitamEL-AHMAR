Projet PressNetwork pour un Organisme de Presse Média

Réalisation d'un site web pour un organisme de presse média, diffusant des informations sur divers événements tels que le sport, la culture, la politique, etc.

Description du projet : Création d'une interface utilisateur conviviale pour permettre aux visiteurs d'accéder rapidement aux dernières nouvelles dans différentes catégories.

Outils utilisés : Technologies modernes en développement web Front-End pour assurer une expérience utilisateur attrayante et réactive.
