Project Management System

A desktop application for task and project tracking

Overview

The Project Management System is a Windows desktop application designed to streamline task tracking, team collaboration, and project delivery. It provides a centralized interface for managing projects, assigning team members, tracking milestones, and maintaining communication through threaded comments.

Developed with C# and Windows Forms, with a Microsoft Access database backend, this application showcases proficiency in desktop software development, database design, and building intuitive user interfaces for productivity tools.

Project Demo Video

Key Features

Task Tracking

Create, assign, and track tasks with priority levels, due dates, status updates, and dependencies.

Team Management

Assign team members to tasks and projects with role-based permissions and workload visibility.

Delivery Tracking

Monitor project milestones, delivery dates, and completion status with visual progress indicators.

Comments System

Threaded comments on tasks for collaboration, with timestamps and user attribution.

Tech Stack

C# Windows Forms Microsoft Access SQL
